## MDX Technical Requirements (from Google Drive docs)

### Tech Stack
- **Backend:** Ruby on Rails + PostgreSQL (Azure, HIPAA compliant)
- **Mobile:** React Native (Vuzix glasses + iOS + Android)
- **Web:** React web client
- **Speech-to-Text:** AssemblyAI (streaming)
- **AI Processing:** OpenAI APIs on Azure
- **Real-time:** WebSocket for live updates

### Target Users
1. Healthcare: Physicians, Nurses, EMTs
2. First Responders: Firefighters, Police
3. Military: Combat medics, Infantry, Medevac, Command
4. Accessibility: Blind users, Dementia patients

### Core Features - Healthcare
- Real-time patient encounter documentation
- EHR integration (Epic, Cerner)
- Voice-to-text transcription
- AI-powered notes and ICD10 coding
- Drug interaction alerts
- Multi-language support (500+ languages)
- Wake word activation

### Core Features - Military
- 9-Line Medevac automation
- Helmet-integrated display
- Secure encrypted communications
- GPS and unit tracking
- Facial recognition
- Thermal/night vision integration

### Core Features - Accessibility
- Audio navigation for blind users
- Facial recognition for dementia patients
- Medication reminders
- Fall detection and emergency alerts
- Smart city integration

### Compliance
- HIPAA (healthcare)
- Military-grade security (DoD)
- ADA/accessibility standards
